# Task Manager (C++)

A simple console-based Task Manager built using **OOP** and **Data Structures** in C++.

## Features
- **Add Tasks:** Create new tasks with a defined priority (1â€“5).  
- **View Tasks:** Display all tasks along with their priority and completion status.

- **Simple Interface:** Intuitive, menu-driven console interface.

- Basic Bubble Sort Algorithm
